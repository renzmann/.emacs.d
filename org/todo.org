* Cheatsheet
+ M-z "zap-to-char"
+ M-% "Query replace"
+ C-x C-v "revert buffer/reload file"
+ C-u C-SPC "Jump back to mark in current buffer"
+ M-@ (same as C-M-SPC) "Expand region forward by a word"
+ M-m "go to first non-whitespace character"
+ C-M-<SPC> and C-M-@ "mark next s-expression"
+ M-r to move cursor between top, bottom, middle
+ M-r reposition center
+ M-0 M-r top
+ M-- M-r bottom
+ M-; to comment stuff
+ C-x p f  find file in project
+ C-x v + to update project from git
+
+ C-d in dired rename/move to accept input literally: http://xahlee.info/emacs/emacs/emacs_icomplete_mode.html
+ C-x x g to revert buffer with fewer confirm messages
+ =M-! find . -name "*.py" -print | etags -= to re-generate project tags
+ When eglot appears to freeze on TRAMP, just hit C-g a bunch of times and it comes back

#+begin_src elisp
;; .dir-locals.el
((python-mode . ((eval . (let* ((dir-local-found (dir-locals-find-file ".dir-locals.el"))
                                (dir-local-path (if (stringp dir-local-found)
                                                    dir-local-found
                                                  (car dir-local-found)))
                                (dynamic-venv-root (expand-file-name ".venv" dir-local-path)))
                           (setq-local python-shell-virtualenv-root dynamic-venv-root))))))
#+end_src
